CSS পরিচিতি: ওয়েবসাইটকে সুন্দর করার জাদুমন্ত্র

আমরা শিখেছি যে, HTML হলো একটি বাড়ির কঙ্কাল বা গঠন। কিন্তু শুধু কঙ্কাল দিয়ে তো আর একটি সুন্দর বাড়ি হয় না! বাড়ির দেয়ালের রঙ কী হবে, জানালাগুলো দেখতে কেমন হবে, দরজাটা কোন ডিজাইনের হবে—এই সব সৌন্দর্যবর্ধনের কাজগুলোই করে CSS। CSS এর পুরো নাম Cascading Style Sheets।

CSS লেখার নিয়মকানুন (Syntax)

CSS দিয়ে ডিজাইন করার জন্য আমাদের একটি নির্দিষ্ট ভাষায় নির্দেশ দিতে হয়। ভাবো, তুমি একজন জাদুকর এবং তোমার HTML এলিমেন্টগুলোকে তুমি তোমার ইচ্ছেমতো পরিবর্তন করতে চাও। এর জন্য তোমাকে একটি জাদুমন্ত্র বা নিয়ম মেনে চলতে হবে।

একটি সম্পূর্ণ CSS নিয়মকে "Rule Set" বলা হয়, যার দুটি প্রধান অংশ থাকে:

  1. সিলেক্টর (Selector): তুমি কোন HTML এলিমেন্টটিকে ডিজাইন করতে চাও, তা খুঁজে বের করার জন্য এটি ব্যবহৃত হয়।
  2. ডিক্লারেশন ব্লক (Declaration Block): এখানে তুমি বলে দাও যে, কী কী পরিবর্তন করতে চাও। এটি দ্বিতীয় বন্ধনী { } দিয়ে লেখা হয়।

চলো, একটি উদাহরণ দিয়ে বুঝি:

h1 { color: blue; font-size: 24px; }
  • সিলেক্টর (Selector): এখানে সিলেক্টর হলো h1। এর মানে হলো, "হে HTML ফাইল, তোমার ভেতরে যত <h1> ট্যাগ আছে, তাদের সবাইকে খুঁজে বের করো!"
  • ডিক্লারেশন (Declaration): দ্বিতীয় বন্ধনীর ভেতরের প্রতিটি লাইন এক একটি ডিক্লারেশন। যেমন, color: blue; একটি ডিক্লারেশন। প্রতিটি ডিক্লারেশনের আবার দুটি অংশ থাকে:
    • প্রোপার্টি (Property): তুমি কী পরিবর্তন করতে চাও। যেমন: color (লেখার রঙ), font-size (লেখার আকার)।
    • ভ্যালু (Value): তুমি পরিবর্তন করে কী করতে চাও। যেমন: blue (নীল), 24px (২৪ পিক্সেল)।

সহজ কথায়: উপরের কোডটি ব্রাউজারকে বলছে, "সব h1 ট্যাগের লেখার রঙ নীল করে দাও এবং তাদের আকার ২৪ পিক্সেল করে দাও।"

CSS লেখার ৩টি পদ্ধতি

একটি HTML ফাইলে CSS যোগ করার তিনটি প্রধান উপায় আছে। প্রত্যেকটির নিজস্ব সুবিধা ও অসুবিধা রয়েছে।

১. ইনলাইন সিএসএস (Inline CSS)

যখন কোনো একটি নির্দিষ্ট ট্যাগের মধ্যেই স্টাইল যোগ করা হয়, তখন তাকে ইনলাইন সিএসএস বলে। এটি style অ্যাট্রিবিউট ব্যবহার করে লেখা হয়।

<h1 style="color: blue; font-size: 30px;">এটি একটি নীল রঙের হেডিং</h1>

২. ইন্টারনাল সিএসএস (Internal CSS)

যখন HTML ফাইলের <head> সেকশনের ভেতরে <style> ট্যাগ ব্যবহার করে CSS লেখা হয়, তখন তাকে ইন্টারনাল সিএসএস বলে।

<head> <style> h1 { color: navy; } </style> </head>

৩. এক্সটারনাল সিএসএস (External CSS) - (সবচেয়ে ভালো পদ্ধতি)

এটি হলো CSS লেখার সবচেয়ে আধুনিক এবং কার্যকরী পদ্ধতি। এখানে, আমরা সম্পূর্ণ আলাদা একটি .css ফাইল তৈরি করি এবং <link> ট্যাগ ব্যবহার করে HTML ফাইলের সাথে যুক্ত করে দিই।

<head> <link rel="stylesheet" href="style.css"> </head>
