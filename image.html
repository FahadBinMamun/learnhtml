ওয়েবপেজে ছবি যোগ করা: ছবিতে কথা বলা

একটি গল্পের বই যদি শুধু লেখা দিয়ে ভর্তি থাকে, কিন্তু কোনো ছবি না থাকে, তাহলে কি পড়তে মজা লাগবে? একদমই না! ওয়েবসাইটও ঠিক তেমনই। ছবি একটি ওয়েবপেজকে প্রাণবন্ত ও আকর্ষণীয় করে তোলে। চলো দেখি, কীভাবে আমরা আমাদের পেজে ছবি যোগ করার জাদু শিখতে পারি।

<img> ট্যাগ: তোমার জাদুর জানালা

ওয়েবসাইটে ছবি যোগ করার জন্য আমরা <img> নামে একটি বিশেষ ট্যাগ ব্যবহার করি। তুমি এটিকে একটি "জাদুর জানালা" ভাবতে পারো, যা দিয়ে আমরা যেকোনো ছবিকে আমাদের ওয়েবপেজে দেখাতে পারি।

src এবং alt: ছবির পরিচয় ও ঠিকানা

আমরা আগেই শিখেছি, src অ্যাট্রিবিউট দিয়ে ছবির ঠিকানা এবং alt অ্যাট্রিবিউট দিয়ে ছবির বর্ণনা দিতে হয়। এই দুটি অ্যাট্রিবিউট ছাড়া <img> ট্যাগ অসম্পূর্ণ।

<img src="images/dog.jpg" alt="একটি বাদামী রঙের কুকুর মাঠে খেলছে">

ছবির আকার নিয়ন্ত্রণ: ছোট থেকে বড়

অনেক সময় ছবিগুলো অনেক বড় বা অনেক ছোট হতে পারে। আমরা HTML বা CSS ব্যবহার করে ছবির আকার নিয়ন্ত্রণ করতে পারি।

১. HTML দিয়ে (সহজ কিন্তু কম কার্যকরী)

width এবং height অ্যাট্রিবিউট ব্যবহার করে ছবির আকার পিক্সেল এককে ঠিক করে দেওয়া যায়।

<img src="cat.jpg" alt="বিড়াল" width="300" height="200">

সমস্যা: এই পদ্ধতিতে ছবির আকার নির্দিষ্ট হয়ে যায়। ফলে, মোবাইল ফোনের মতো ছোট স্ক্রিনে ছবিটি কেটে যেতে পারে।

২. CSS দিয়ে (স্মার্ট এবং রেসপন্সিভ পদ্ধতি)

সবচেয়ে ভালো উপায় হলো CSS ব্যবহার করা। max-width: 100% ব্যবহার করলে ছবিটি তার কন্টেইনারের চেয়ে বড় হবে না এবং স্ক্রিনের আকার অনুযায়ী নিজে থেকেই ছোট-বড় হবে।

<style> .responsive-image { max-width: 100%; height: auto; } </style> <img src="cat.jpg" class="responsive-image">

ব্যাকগ্রাউন্ডে ছবি যোগ করা

<img> ট্যাগ দিয়ে আমরা যে ছবি যোগ করি, তা হলো একটি কন্টেন্ট। কিন্তু যদি আমরা কোনো লেখার পেছনে বা পুরো পেজের ব্যাকগ্রাউন্ডে ছবি দিতে চাই, তখন আমাদের CSS এর সাহায্য নিতে হয়।

এই কাজটি background-image প্রোপার্টি দিয়ে করা হয়।

<div class="hero-section"> <h1>আমাদের ক্লাবে স্বাগতম</h1> </div> <style> .hero-section { /* ছবির ঠিকানা এখানে দিতে হবে */ background-image: url('images/background.jpg'); background-size: cover; /* ছবিটি যেন পুরো জায়গা জুড়ে থাকে */ padding: 50px; color: white; } </style>

আউটপুট কেমন হবে:

আমাদের ক্লাবে স্বাগতম

নতুনদের জন্য বিশেষ টিপস

  • ফাইল পাথ ঠিক রাখা: src="images/cat.jpg" লেখার মানে হলো, তোমার HTML ফাইলটি যেখানে আছে, তার পাশেই `images` নামে একটি ফোল্ডার আছে এবং তার ভেতরে `cat.jpg` ছবিটি আছে। ঠিকানা ভুল হলে ছবি দেখা যাবে না।
  • ছবির আকার কমানো: ওয়েবসাইটে দেওয়ার আগে ছবির ফাইল সাইজ কমিয়ে (Optimize) নেওয়া ভালো। এতে ওয়েবসাইট দ্রুত লোড হয়।
  • সঠিক ফরম্যাট: সাধারণ ছবির জন্য .jpg, লোগো বা স্বচ্ছ ব্যাকগ্রাউন্ডের জন্য .png এবং ছোট অ্যানিমেশনের জন্য .gif ব্যবহার করা ভালো।
