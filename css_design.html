CSS দিয়ে ডিজাইন: তোমার সৃজনশীলতার প্রকাশ

এখন আমরা CSS এর জাদুর বাক্সের ভেতরে প্রবেশ করব। এখানে আমরা শিখব কীভাবে বিভিন্ন CSS প্রোপার্টি ব্যবহার করে একটি সাদামাটা ওয়েবপেজকে রঙিন এবং আকর্ষণীয় করে তোলা যায়।

রঙের খেলা: color এবং background-color

সবচেয়ে মজার কাজ হলো রঙ নিয়ে খেলা করা। CSS এ দুটি প্রধান প্রোপার্টি দিয়ে রঙ পরিবর্তন করা হয়:

  • color: এটি দিয়ে লেখার রঙ পরিবর্তন করা হয়।
  • background-color: এটি দিয়ে যেকোনো এলিমেন্টের পেছনের বা ব্যাকগ্রাউন্ডের রঙ পরিবর্তন করা হয়।
.my-box { background-color: navy; /* الخلفية রঙ নেভি ব্লু */ color: white; /* লেখার রঙ সাদা */ padding: 20px; }

আউটপুট:

এই বক্সটির ব্যাকগ্রাউন্ড নেভি এবং লেখা সাদা।

লেখার সাজসজ্জা: ফন্ট এবং টেক্সট

ওয়েবসাইটের লেখা দেখতে কেমন হবে, তা নিয়ন্ত্রণ করা খুবই জরুরি।

  • font-family: লেখার ফন্ট পরিবর্তন করে।
  • font-size: লেখার আকার ছোট বা বড় করে।
  • font-weight: লেখাকে মোটা (bold) বা সাধারণ (normal) করে।
  • text-align: লেখাকে বামে, ডানে বা মাঝে সারিবদ্ধ করে।
.title { font-family: Verdana, sans-serif; font-size: 30px; font-weight: bold; text-align: center; }
