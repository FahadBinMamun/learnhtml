HTML অ্যাট্রিবিউট: ট্যাগের শক্তি বৃদ্ধি

অ্যাট্রিবিউট হলো HTML ট্যাগের সুপারপাওয়ার! এগুলো ট্যাগকে অতিরিক্ত তথ্য দেয় এবং তার সাধারণ আচরণের বাইরে নতুন নতুন কাজ করার ক্ষমতা দেয়।

অ্যাট্রিবিউট কী? একটি সহজ উদাহরণ

ধরো, <p> ট্যাগটি হলো "একটি মানুষ"। এই তথ্যটুকু যথেষ্ট নয়। মানুষটির পরিচয় কী? তার নাম কী? সে কোন ক্লাসে পড়ে?

অ্যাট্রিবিউট এই অতিরিক্ত পরিচয়গুলোই দেয়। যেমন, আমরা যদি বলি, "মানুষটি একজন ছাত্র, তার নাম করিম", তাহলে তার সম্পর্কে ধারণা পরিষ্কার হয়। HTML-এ আমরা এটা এভাবে লিখি:

<p class="student" id="karim">আমি একজন ছাত্র।</p>

এখানে class এবং id হলো দুটি অ্যাট্রিবিউট, যা <p> ট্যাগটিকে আরও নির্দিষ্ট করে তুলেছে।

অ্যাট্রিবিউটের গঠন

প্রতিটি অ্যাট্রিবিউটের দুটি প্রধান অংশ থাকে এবং এটি সবসময় ট্যাগের শুরুর অংশে (Opening Tag) লেখা হয়।

  1. Name (নাম): এটি অ্যাট্রিবিউটের পরিচয়। যেমন: href, src, class।
  2. Value (মান): এটি অ্যাট্রিবিউটের নির্দিষ্ট মান, যা সবসময় ডাবল কোটেশন (" ") বা সিঙ্গেল কোটেশন (' ') এর ভেতরে লেখা হয়।

গঠন: name="value"

অ্যাট্রিবিউটের প্রকারভেদ

অ্যাট্রিবিউট মূলত দুই প্রকারের হয়:

১. গ্লোবাল অ্যাট্রিবিউট (Global Attributes)

এই অ্যাট্রিবিউটগুলো প্রায় সব HTML ট্যাগের সাথেই ব্যবহার করা যায়। এগুলোকে HTML-এর "কমন সুপারপাওয়ার" বলা যেতে পারে।

  • id: একটি এলিমেন্টকে একটি ইউনিক বা অদ্বিতীয় নাম দেয়। একটি পেজে একই নামের id একাধিকবার ব্যবহার করা যায় না। এটা অনেকটা ক্লাসের রোল নম্বরের মতো।
    <h2 id="main-title">মূল শিরোনাম</h2>
  • class: একাধিক এলিমেন্টকে একই গ্রুপে ফেলার জন্য class ব্যবহার করা হয়। যেমন, একটি পেজের সব বাটনকে একই ডিজাইন দেওয়ার জন্য তাদের সবাইকে একই class দেওয়া যেতে পারে।
    <button class="btn-primary">ক্লিক করুন</button> <button class="btn-primary">আবার ক্লিক করুন</button>
  • style: এটি দিয়ে সরাসরি একটি ট্যাগের মধ্যে CSS কোড লেখা যায় (একে Inline CSS বলে)। দ্রুত কোনো পরিবর্তনের জন্য এটি সুবিধাজনক হলেও, বড় প্রজেক্টে এটি ব্যবহার করা অনুচিত।
    <p style="color: blue; font-size: 20px;">এই লেখাটি নীল।</p>
  • title: কোনো এলিমেন্টের উপর মাউস নিয়ে গেলে (Hover করলে) যে ছোট টেক্সট বক্সটি দেখা যায়, তা এই অ্যাট্রিবিউট দিয়ে ঠিক করা হয়।
    <p title="এটি একটি প্যারাগ্রাফ">আমার সম্পর্কে জানতে মাউস আনুন।</p>

২. নির্দিষ্ট অ্যাট্রিবিউট (Element-Specific Attributes)

এই অ্যাট্রিবিউটগুলো শুধুমাত্র নির্দিষ্ট কিছু ট্যাগের সাথেই কাজ করে।

  • href: এটি শুধু <a> (অ্যাঙ্কর) ট্যাগের সাথে কাজ করে এবং লিঙ্কের গন্তব্য নির্ধারণ করে।
    <a href="https://google.com">Google</a>
  • src: এটি <img>, <video>, <iframe> ট্যাগের সাথে কাজ করে এবং ফাইলের উৎস (Source) বলে দেয়।
    <img src="photo.jpg">
  • alt: এটি শুধু <img> ট্যাগের সাথে ব্যবহৃত হয়। ছবি লোড না হলে এই লেখাটি দেখায়।
    <img src="photo.jpg" alt="একটি সুন্দর প্রাকৃতিক দৃশ্য">
  • disabled: এটি <button> বা <input> ট্যাগকে নিষ্ক্রিয় করে দেয়।
    <button disabled>এই বাটন কাজ করবে না</button>

নতুনদের জন্য বিশেষ টিপস

  • সবসময় ছোট হাতের অক্ষর ব্যবহার করুন: যদিও HTML অ্যাট্রিবিউটের নাম বড় হাতের অক্ষরে লিখলেও কাজ করে, কিন্তু স্ট্যান্ডার্ড নিয়ম হলো ছোট হাতের অক্ষর (lowercase) ব্যবহার করা।
  • কোটেশন ভুলবেন না: অ্যাট্রিবিউটের ভ্যালু সবসময় ডাবল (" ") বা সিঙ্গেল (' ') কোটেশনের ভেতরে রাখুন। এটি একটি ভালো অভ্যাস।
  • class বনাম id: যখন অনেকগুলো এলিমেন্টকে একই রকম ডিজাইন দিতে চাইবেন, তখন class ব্যবহার করুন। আর যখন একটি মাত্র নির্দিষ্ট এলিমেন্টকে ধরতে চাইবেন (বিশেষ করে JavaScript দিয়ে), তখন id ব্যবহার করুন।
  • অর্থপূর্ণ নাম দিন: class বা id এর নাম এমনভাবে দিন যেন তা দেখেই বোঝা যায় এলিমেন্টটি কী কাজ করছে। যেমন: class="header-title" বা id="login-form"।
